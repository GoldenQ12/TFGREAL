# Full Stack Project

This project consists of a **frontend** and **backend** setup using Node.js and npm. Follow the instructions below to get started with development.

## Prerequisites

- Node.js (v14 or later)
- npm (comes with Node.js)

---

## Installation

- cd backend
- npm install
- cd frontend
- npm install

## Launch the App

- npm run dev
